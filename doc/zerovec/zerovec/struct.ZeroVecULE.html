Struct zerovec::zerovec::ZeroVecULE[−][src]

#[repr(transparent)]
pub struct ZeroVecULE<T: AsULE>(pub [T::ULE]);
Expand description

A type similar to ZeroVec that can be used inside VarZeroVec<T>.

This essentially allows for the construction of zero-copy types isomorphic to Vec<Vec<T>> by instead using VarZeroVec<ZeroVecULE<T>>.

See the [VarZeroVec] docs for an example.
